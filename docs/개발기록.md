**이번 주 배운 것**
- 네임스페이스 ≠ 어셈블리 (`Unity.Transforms`/`Unity.Rendering`은 각각 `Unity.Entities`/`Unity.Entities.Graphics`가 제공).
- Managed(class) vs Unmanaged(struct) 컴포넌트 — 항상 struct. class는 GC·Burst 불가·캐시 미스에 6.5 deprecated.
- 베이킹/SubScene — GameObject authoring → 엔티티 변환.
- `uloop compile` 초록불이어도 설계는 틀릴 수 있다(빈 어셈블리·private 필드·class 컴포넌트 전부 통과) → 컴파일 ≠ 검증.
